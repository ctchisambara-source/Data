# Excel Data

## Project Overview
This project was undertaken during a Data Technician bootcamp to build essential skills in Microsoft Excel, including data cleaning, transformation, analysis, and visualization. By working with retail and sales datasets, it highlights competence in using formulas, PivotTables, sorting and filtering data, applying categorization logic, and creating charts.

## Skills
During this Excel bootcamp work, I developed practical skills in:
	•	Filtering and sorting data to quickly organise information.
	•	Using formulas like SUM and AVERAGE to calculate totals and averages.
	•	Creating and customising PivotTables to summarise large datasets by categories like county, product, or demographic groups.
	•	Using the SWITCH function to categorise data dynamically based on conditions.
	•	Exploring datasets visually with charts to identify patterns and trends.
	•	Gaining hands-on experience with data cleaning, transformation, and analysis, making raw data easier to interpret and present.
